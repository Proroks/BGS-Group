## Запуск приложения

Заупустить можно на локальной машине или через Docker 

#### Локально

```bash
$ npm install

# локальный сервер с hot reload на localhost:3000
$ npm run dev

```
#### Через Docker

```bash
# зупуск
$ docker-compose up
```
Будет доступен localhost:3000
